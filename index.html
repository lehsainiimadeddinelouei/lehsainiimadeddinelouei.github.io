Hardware

SMART GLASSES

These smart glasses translate ASL into spoken language using an onboard Raspberry Pi Zero, a camera module, mini speakers, and a micro projector — offering real-time translation both online and offline.

01
Capturing Input
Camera module captures hand gestures at 30fps via Raspberry Pi Zero.
02
Sending to Server
Frames are compressed and sent to the inference API over WiFi.
03
AI Processing
MediaPipe + classifier model identifies hand landmarks and predicts letters.
04
Audio Output
Text-to-speech converts translated text into audio through mini speakers.
05
Offline Mode
On-device fallback model activates when no network is available.
